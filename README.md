# Cómo ejecutar
1. Crear virtual environment: `python -m venv .venv`
2. Activar: `.venv\Scripts\activate`
3. Instalar dependencias: `pip install -r requirements.txt`
4. Ejecutar: `python main.py <makespan_objetivo>`
